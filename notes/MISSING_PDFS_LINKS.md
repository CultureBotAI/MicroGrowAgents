## Download Instructions

### Method 1: Browser Download (Recommended)

1. **Connect to LBL Network** (VPN if remote)
2. **Open each DOI link** in your browser
3. **Click "Download PDF"** button on publisher page
4. **Save to** `/tmp/microgrow_pdfs/` with the exact filename shown above

### Method 2: Command Line (macOS/Linux)

If you have authenticated access configured:

```bash
# Create directory
mkdir -p /tmp/microgrow_pdfs

# Download using wget or curl (if cookies/auth are configured)
# Example for one DOI:
wget -O /tmp/microgrow_pdfs/10.1021_bi00866a011.pdf \
  "https://pubs.acs.org/doi/pdf/10.1021/bi00866a011"
```

### Method 3: Library Document Delivery

Contact LBL Library for assistance:
- **Email:** library@lbl.gov
- **Request:** "PDF for DOI: {doi}"
- Typically fulfilled within 1-2 business days
