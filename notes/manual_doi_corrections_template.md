# Manual DOI Correction Guide

## Instructions

For each invalid DOI below:
1. Note the component context (which chemicals use this DOI)
2. Use the search hint to find the correct paper
3. Fill in the CORRECTED_DOI field
4. Add notes about the paper (title, journal, year)

When done, save this file and run the correction script to apply changes.
